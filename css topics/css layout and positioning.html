CSS Basics - Layout and Positioning

Controlling Layout with Display Property

The `display` property in CSS determines the layout behavior of elements. Here are some examples:

Block-level Elements

Elements that generate a block-level box and typically start on a new line.

.container {
    display: block;
}

Inline Elements

Elements that generate an inline box and do not start on a new line.

span {
    display: inline;
}

Inline-Block Elements

Elements that generate a block box that will be flowed with surrounding content as if it were an inline box.

.button {
    display: inline-block;
}

Positioning Elements

Positioning elements allows you to control their placement on the page. Common properties include `position`, `float`, and `flexbox`:

Position Property

Defines the positioning method of an element.

.absolutePosition {
    position: absolute;
    top: 10px;
    left: 20px;
}

Float Property

Specifies whether an element should be placed to the left or right of its container.

.floatLeft {
    float: left;
}

Flexbox

A layout model that allows you to design complex layouts more efficiently.

.flexContainer {
    display: flex;
    justify-content: space-between;
}
