CSS Basics - Box Model

Understanding the Box Model

The box model is a fundamental concept in CSS that defines the structure of an HTML element. It consists of the following components:

  • Content
  • Padding
  • Border
  • Margin

Box Model Properties

Adjusting the layout of elements is done by manipulating the box model properties. Here are some examples:

Width and Height

Setting the width and height of an element, including content, padding, and border.

.box {
    width: 200px;
    height: 100px;
}

Padding

Adding internal space around the content.

.box {
    padding: 20px;
}

Border

Defining the border around the content and padding.

.box {
    border: 2px solid #333;
}

Margin

Creating space outside the element.

.box {
    margin: 10px;
}
