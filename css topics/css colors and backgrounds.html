CSS Basics - Colors and Backgrounds

Specifying Colors

Colors in CSS can be specified using various formats. Here are some examples:

Color Names

Using predefined color names.

.heading {
    color: red;
}

Hexadecimal Values

Using hexadecimal values for more precise color control.

.paragraph {
    color: #336699;
}

RGB Values

Defining colors using RGB values.

.background {
    background-color: rgb(255, 0, 0);
}

HSL Values

Applying colors using HSL values.

.link {
    color: hsl(120, 100%, 50%);
}

Background Colors and Images

Enhancing the appearance of elements with background colors and images:

Background Color

Setting a background color for an element.

.container {
    background-color: #f0f0f0;
}

Background Image

Adding a background image to an element.

.header {
    background-image: url('background.jpg');
    background-size: cover;
}
