CSS Basics - Transitions and Animations

Transitions

Transitions in CSS allow you to smoothly change property values. Here's an example:

.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #007BFF;
}

Animations with @keyframes

Animations are created using the `@keyframes` rule. Here's a simple animation example:

@keyframes bounce {
    0%, 20%, 50%, 80%, 100% {
        transform: translateY(0);
    }
    40% {
        transform: translateY(-30px);
    }
    60% {
        transform: translateY(-15px);
    }
}

.bouncingBall {
    animation: bounce 1s infinite;
}
