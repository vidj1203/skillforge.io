CSS Basics - Selectors and Declarations

Selectors in CSS

CSS selectors are patterns used to select and style HTML elements. They determine which elements in the HTML document the styles should be applied to.

Universal Selector

The universal selector `*` selects all elements on the page.

* {
    margin: 0;
    padding: 0;
}

Type Selector

Selects all instances of a given HTML element type.

h1 {
    color: #007BFF;
}

Class Selector

Selects elements with a specific class attribute.

.myClass {
    font-weight: bold;
}

Declarations in CSS

Declarations are used to apply styles to selected elements. Each declaration consists of a property and a value.

selector {
    property: value;
}

Example Declarations

Applying styles to the selected elements.

h2 {
    color: #555;
    font-size: 18px;
}

.container {
    width: 80%;
    margin: 0 auto;
}
