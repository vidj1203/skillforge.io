CSS Basics - CSS Grid

Exploring CSS Grid Layout

CSS Grid is a powerful layout system that allows you to create complex and structured designs. Here's an introduction:

Defining a Grid Container

Turn an element into a grid container:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: auto;
}

Grid Items

Define how items should be placed within the grid:

.item {
    grid-column: span 2;
    grid-row: 1;
}

Creating Grid-Based Layouts

Use CSS Grid to structure layouts efficiently:

Grid Template Areas

Create named areas for layout:

.container {
    display: grid;
    grid-template-areas:
        'header header header'
        'main main sidebar'
        'footer footer footer';
}

.header {
    grid-area: header;
}

.main {
    grid-area: main;
}

.sidebar {
    grid-area: sidebar;
}

.footer {
    grid-area: footer;
}
