CSS Basics - Properties and Values

Common CSS Properties

Several CSS properties are commonly used to style HTML elements. Here are a few examples:

Color Property

Defines the text color.

h1 {
    color: #007BFF;
}

Font Size Property

Sets the size of the font.

p {
    font-size: 16px;
}

Margin and Padding Properties

Control the spacing around elements.

.container {
    margin: 10px;
    padding: 20px;
}

Assigning Values to Properties

Values are assigned to properties to define how the style should be applied. Values can be numeric, color codes, keywords, etc.

Numeric Values

Assigning numeric values to properties like width or font size.

.container {
    width: 80%;
    font-size: 18px;
}

Color Values

Using color values to define text or background colors.

body {
    background-color: #f0f0f0;
    color: #333;
}

Keyword Values

Using keywords for properties like `display`.

.section {
    display: flex;
}
