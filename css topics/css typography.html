CSS Basics - Typography

Text Styling Properties

Typography in CSS involves styling text to enhance readability and aesthetics. Here are some text styling properties:

Font Family

Specifying the font family for text.

body {
    font-family: 'Arial', sans-serif;
}

Font Size

Setting the size of the text.

h1 {
    font-size: 36px;
}

Font Weight

Adjusting the thickness of the text.

.boldText {
    font-weight: bold;
}

Line Height

Defining the space between lines of text.

p {
    line-height: 1.5;
}

Text Alignment and Decoration

Controlling the alignment and decoration of text:

Text Alignment

Aligning text within its container.

.textCenter {
    text-align: center;
}

Text Decoration

Adding decoration to text, such as underlining or striking through.

.link {
    text-decoration: underline;
}
