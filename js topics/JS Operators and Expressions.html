JavaScript Basics - Operators and Expressions

Arithmetic Operators

Arithmetic operators perform mathematical calculations. Here are some common ones:

Addition (+):

let sum = 5 + 3; // Result: 8

Subtraction (-):

let difference = 10 - 4; // Result: 6

Multiplication (*):

let product = 2 * 5; // Result: 10

Division (/):

let quotient = 20 / 4; // Result: 5

Comparison Operators

Comparison operators compare values and return a boolean result. Examples include:

Equal to (==):

let isEqual = 5 == 5; // Result: true

Not equal to (!=):

let isNotEqual = 10 != 5; // Result: true

Greater than (>), Less than (<):

let isGreater = 8 > 3; // Result: true
let isLess = 4 < 7; // Result: true

Logical Operators

Logical operators perform logical operations. Examples include:

AND (&&):

let bothTrue = true && true; // Result: true

OR (||):

let eitherTrue = true || false; // Result: true

NOT (!):

let notTrue = !false; // Result: true
