JavaScript Basics - DOM Manipulation

Interacting with the Document Object Model (DOM)

The DOM represents the structure of a document as a tree of objects. Here's how you can manipulate it:

Accessing Elements by ID:

let myElement = document.getElementById('myElement');

Updating Text Content:

myElement.textContent = 'New content';

Changing Styles:

myElement.style.color = 'red';
myElement.style.fontSize = '18px';

Modifying HTML Structure

You can add, remove, or modify HTML elements using DOM manipulation:

Adding Elements:

let newElement = document.createElement('p');
newElement.textContent = 'This is a new paragraph.';
document.body.appendChild(newElement);

Removing Elements:

let oldElement = document.getElementById('oldElement');
oldElement.parentNode.removeChild(oldElement);
