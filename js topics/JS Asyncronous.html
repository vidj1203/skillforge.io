JavaScript Basics - Asynchronous JavaScript

Understanding Asynchronous Programming

Asynchronous JavaScript allows you to execute code without blocking the main thread. Here's an overview of asynchronous programming:

Callbacks:

function fetchData(callback) {
    // Simulating an asynchronous operation
    setTimeout(function() {
        let data = 'Async data';
        callback(data);
    }, 1000);
}

fetchData(function(result) {
    console.log(result); // Outputs: Async data
});

Using Promises:

function fetchData() {
    return new Promise(function(resolve, reject) {
        // Simulating an asynchronous operation
        setTimeout(function() {
            let data = 'Async data';
            resolve(data);
        }, 1000);
    });
}

fetchData().then(function(result) {
    console.log(result); // Outputs: Async data
});

Async and Await

Async and Await provide a more readable and synchronous-like way to work with asynchronous code:

Using Async and Await:

async function fetchData() {
    return new Promise(function(resolve, reject) {
        // Simulating an asynchronous operation
        setTimeout(function() {
            let data = 'Async data';
            resolve(data);
        }, 1000);
    });
}

async function getData() {
    let result = await fetchData();
    console.log(result); // Outputs: Async data
}

getData();
