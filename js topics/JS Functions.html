JavaScript Basics - Functions

Defining and Calling Functions

Functions are blocks of reusable code. Here's how you define and call them:

Function Definition:

function greet() {
    console.log("Hello, World!");
}

Function Call:

greet(); // Output: Hello, World!

Functions can take parameters and return values, allowing for more flexibility.

Parameters and Return Values

Parameters are variables passed into a function, and return values are the results of a function's execution:

Function with Parameters:

function greetUser(name) {
    console.log("Hello, " + name + "!");
}

greetUser("John"); // Output: Hello, John!

Function with Return Value:

function addNumbers(a, b) {
    return a + b;
}

let result = addNumbers(5, 3);
console.log(result); // Output: 8
