JavaScript Basics - Control Flow

Conditional Statements (if, else if, else)

Conditional statements allow you to make decisions in your code. Here's an introduction to them:

if Statement:

let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
}

else if Statement:

let time = 14;
if (time < 12) {
    console.log("Good morning!");
} else if (time < 18) {
    console.log("Good afternoon!");
} else {
    console.log("Good evening!");
}

else Statement:

let isRaining = false;
if (isRaining) {
    console.log("Remember to take an umbrella!");
} else {
    console.log("Enjoy the weather!");
}

Loops (for, while)

Loops are used for repetitive tasks. Here are examples of 'for' and 'while' loops:

for Loop:

for (let i = 1; i <= 5; i++) {
    console.log("Iteration " + i);
}

while Loop:

let counter = 0;
while (counter < 3) {
    console.log("Count: " + counter);
    counter++;
}
