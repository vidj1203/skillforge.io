JavaScript Basics - Objects

Understanding Objects and Properties

Objects in JavaScript are used to store and organize data. Here's how you define and work with objects:

Object Definition:

let person = {
    name: 'John',
    age: 25,
    gender: 'Male'
};

Accessing Object Properties:

let personName = person.name; // Accesses the name property (John)

Modifying Object Properties:

person.age = 26; // Changes the age property to 26

Object Methods and Accessing Values

Objects can contain methods (functions) and values. Here's how you work with them:

Object with Method:

let car = {
    brand: 'Toyota',
    model: 'Camry',
    startEngine: function() {
        console.log('Engine started!');
    }
};

car.startEngine(); // Outputs: Engine started!

Accessing Values Dynamically:

let key = 'brand';
let carBrand = car[key]; // Accesses the value of the 'brand' property (Toyota)
