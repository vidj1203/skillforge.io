JavaScript Basics - Variables and Data Types

Variables

In JavaScript, variables are used to store and manage data. Here's how you declare and use them:

Declaration:

let variableName; // Declaration

Assignment:

variableName = 10; // Assignment

Initialization:

let anotherVariable = "Hello, World!"; // Initialization

Variables can hold various types of data, which brings us to the next topic.

Data Types

JavaScript supports several data types, each serving a different purpose. The most common ones include:

1. String:

let name = "John";

2. Number:

let age = 25;

3. Boolean:

let isStudent = true;

4. Undefined:

let undefinedVariable;

5. Null:

let nullValue = null;

Understanding these data types is crucial for working with variables effectively.
