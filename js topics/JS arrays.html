JavaScript Basics - Arrays

Creating and Manipulating Arrays

Arrays are used to store and retrieve collections of data. Here's how you create and manipulate arrays:

Array Creation:

let fruits = ['Apple', 'Banana', 'Orange'];

Accessing Elements:

let firstFruit = fruits[0]; // Accesses the first element (Apple)

Modifying Elements:

fruits[1] = 'Grapes'; // Changes the second element to Grapes

Array Methods

JavaScript provides various methods to manipulate arrays efficiently:

push:

fruits.push('Mango'); // Adds Mango to the end of the array

pop:

let lastFruit = fruits.pop(); // Removes and returns the last element (Mango)

shift:

let firstRemoved = fruits.shift(); // Removes and returns the first element (Apple)

unshift:

fruits.unshift('Kiwi'); // Adds Kiwi to the beginning of the array
