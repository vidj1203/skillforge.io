Lists

Lists in HTML allow you to organize and structure information. There are two main types of lists: ordered lists and unordered lists.

Ordered Lists (<ol>)

An ordered list is a numbered list where each item is preceded by a number. You can create an ordered list using the <ol> tag. Each list item is defined by the <li> tag. Here's an example:

<ol>
<li>Item 1</li>
<li>Item 2</li>
</ol>

This will result in:

  1. Item 1
  2. Item 2

Unordered Lists (<ul>)

An unordered list is a bulleted list where each item is preceded by a bullet point. You can create an unordered list using the <ul> tag. Like ordered lists, each list item is defined by the <li> tag. Example:

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

This will result in:

  • Item 1
  • Item 2
