Basic CSS Integration

Introducing the concept of Cascading Style Sheets (CSS) and how to link external stylesheets to HTML documents for styling and layout purposes.

What is CSS?

Cascading Style Sheets (CSS) is a stylesheet language used for describing the presentation of a document written in HTML or XML. CSS enhances the visual presentation of web pages by defining styles such as colors, fonts, and layouts.

Linking External Stylesheet

To integrate CSS with HTML, you can link an external stylesheet using the <link> tag in the <head> section of your HTML document.

<link rel="stylesheet" href="styles.css">

Internal CSS

You can also include CSS styles internally within the <style> tag in the <head> section. This is useful for small projects or when specific styles are needed for a single page.

<style>
    body {
        background-color: #f0f0f0;
        font-family: 'Arial', sans-serif;
    }
    /* Other styles go here */
</style>
