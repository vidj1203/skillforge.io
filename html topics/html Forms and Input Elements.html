HTML Forms and Input Elements

Forms and Input Elements

Building forms allows users to interact with your website. Let's explore the <form> tag and various input elements.

Basic Form Structure

Use the <form> tag to wrap your form elements. Add attributes like action and method for specifying where the form data should be sent and how.

<form action="/submit" method="post">
    
</form>

Text Input (<input type="text">)

Create a single-line text input field with the <input> tag and set the type attribute to "text".

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Password Input (<input type="password">)

For sensitive information like passwords, use the password type.

<label for="password">Password:</label>
<input type="password" id="password" name="password">

Radio Buttons (<input type="radio">)

Allow users to choose one option from a list using radio buttons.

<input type="radio" id="male" name="gender" value="male">
<label for="male">Male</label>

<input type="radio" id="female" name="gender" value="female">
<label for="female">Female</label>

Checkboxes (<input type="checkbox">)

Enable users to select multiple options with checkboxes.

<input type="checkbox" id="subscribe" name="subscribe" value="yes">
<label for="subscribe">Subscribe to newsletter</label>

Submit Button (<input type="submit">)

End the form with a submit button to send the data to the server.

<input type="submit" value="Submit">
