HTML Semantics

Semantic HTML

Understanding the importance of semantic HTML tags for better page structure and accessibility.

Header Tag (<header>)

The <header> tag is used to define a header section for a document or a section. It often contains a heading element and other introductory content.

<header>
    <h1>Page Title</h1>
    <p>Welcome to our website!</p>
</header>

Navigation Tag (<nav>)

The <nav> tag is used to define a navigation menu. It typically contains a list of links for navigation purposes.

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>

Section Tag (<section>)

The <section> tag is used to define a section in a document. It is often used to group related content together and can have its own heading.

<section>
    <h2>Introduction</h2>
    <p>This section contains introductory content.</p>
</section>

Article Tag (<article>)

The <article> tag is used to define a self-contained piece of content. It can represent a blog post, a news article, a forum post, etc.

<article>
    <h2>Blog Post Title</h2>
    <p>Content of the blog post.</p>
</article>

Footer Tag (<footer>)

The <footer> tag is used to define a footer for a document or a section. It often contains metadata, copyright information, or links to related resources.

<footer>
    <p>Copyright © 2024 SkillForge</p>
</footer>

Using semantic HTML tags improves both the structure of your page and its accessibility. These tags help browsers and assistive technologies understand the content better.
