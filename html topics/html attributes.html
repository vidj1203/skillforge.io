HTML Attributes

Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and come in name/value pairs. Let's explore some common attributes:

  • href: Used in the <a> tag for hyperlinks. It specifies the URL the link goes to. Example: <a href="https://www.example.com" target="_blank">Visit Example</a>.
  • src: Used in the <img> tag for images. It specifies the source (file path or URL) of the image. Example: <img src="image.jpg" alt="Description of the image">.
  • alt: Used in the <img> tag for alternative text. It provides a text description of the image. Example: <img src="image.jpg" alt="Description of the image">.

Paired Tags with Attributes

Attributes are often used with paired tags. Here's an example with the <a> and <img> tags:

<a href="https://www.example.com" target="_blank">Visit Example</a>
<img src="image.jpg" alt="Description of the image">
