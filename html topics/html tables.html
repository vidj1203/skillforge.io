Tables in HTML

Understanding the HTML tags related to tables and how they work to structure tabular data.

Table Tag (<table>)

The <table> tag is used to define a table in HTML. It acts as a container for other table-related tags.

<table>
    
</table>

Table Row Tag (<tr>)

The <tr> tag defines a table row. It is used to group the content of a row in a table.

<tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
</tr>

Table Header Tag (<th>)

The <th> tag defines a header cell in a table. It is used to represent header information in the table.

<tr>
    <th>Header 1</th>
    <th>Header 2</th>
</tr>

Table Data Tag (<td>)

The <td> tag defines a standard cell in a table. It is used to represent data in the table.

<tr>
    <td>Data 1</td>
    <td>Data 2</td>
</tr>

Table Caption Tag (<caption>)

The <caption> tag is used to add a caption to a table. It should be placed immediately after the <table> tag.

<table>
    <caption>Table Caption</caption>
    
</table>

Tables in HTML are essential for displaying and structuring tabular data. Use these tags to create well-organized tables for better data presentation.
