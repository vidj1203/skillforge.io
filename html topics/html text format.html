HTML Text Formatting

Text Formatting

HTML provides tags for basic text formatting. These tags allow you to style and structure your text. Let's explore some common formatting tags:

  • <strong>: Used to make text bold. Example: <p>This is <strong>bold</strong> text.</p>
  • <em>: Used to make text italic. Example: <p>This is <em>italic</em> text.</p>
  • <u>: Used to underline text. Example: <p>This is <u>underlined</u> text.</p>
  • <br>: Used to create a line break. It is a self-closing tag and doesn't have a closing tag. Example: <p>This line breaks here <br>and continues on the next line.</p>

Text Formatting Examples

Here are some examples of text formatting tags in action:
