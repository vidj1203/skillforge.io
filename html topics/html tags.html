Html Tags

Headings

Headings are like titles for your content. Use <h1> for the main title, <h2> for subheadings, <h3> for sub-subheadings, and until <h6>. Each heading level represents a different level of importance.

Paragraphs

Paragraphs group text together. Use the <p> tag to create a new paragraph.

Links

Links take you to other pages. Create a link with the <a> tag. The URL goes inside the href attribute. For example: <a href="https://example.com">Visit Example</a>

Images

Show pictures on your page using the <img> tag. The image file path goes in the src attribute. For example: <img src="image.jpg" alt="Description of the image">

Lists

Lists organize information. Use <ul> for a bulleted list, <ol> for a numbered list, and <li> for each list item. For example:

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

Paired Tags

Many HTML tags come in pairs, with an opening tag and a closing tag. The content goes between them. For example, <b> for bold text and </b> to close it: <b>This is bold text</b>.
